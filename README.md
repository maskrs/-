## 挂机模式
★&ensp;角色自动轮换模式：自动轮换人类/屠夫角色，若选择此模式，运行工具后输入你需要轮换的角色数量（比如：你想刷20个屠夫，让他们轮换挂机，就输入20）；  

&ensp;&ensp;&ensp;●&ensp;选择 *角色自动轮换* 模式，请选择本阵营 **第一个角色** *【德怀特/设陷者】* 并 ***装配好所有轮换角色的技能*** 再开始运行工具！【推荐屠夫使用】  

&ensp;&ensp;&ensp;●&ensp;【子模式】单独行角色模式  

&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;○&ensp;选择单独行角色模式，选择 *该行* 的 **第一个角色** 再运行工具！【推荐屠夫使用】

&emsp;&emsp;&emsp;&emsp;<img src="https://github.com/maskrs/DBD_AFK_TOOL/blob/main/image-foder/%E8%AF%B4%E6%98%8E1.png" width="600px">

★&ensp;单独角色挂机模式：若选择此模式，将单独运行所选定的角色  

&ensp;&ensp;&ensp;●&ensp;选择 *单独角色挂机* 模式，请选择 *要挂机的角色* 后再运行工具！【人类挂机只推荐此模式；屠夫推荐使用】
## 行为模式
&emsp;`挂机要先搞清楚目的。挂机应以裂隙、裂片为主，不以血点为目的。`  

- 随机模式：以随机移动闪避人类道具，最大程度规避左键，以技能为主，逼迫人类尽快出门，从而提高效率。配合怨气冲天和光明之眼可使得对局效率大大提升。【全屠夫技能适配】  

- 固定模式：固定移动方式，不闪避，以左键攻击为主，特殊能力和技能为辅。  
## 角色设置
1. 确保角色 *【技能格】* 携带 *【常亮】* 技能 ，技能等级不限！但为了稳定性强烈建议携带 **四个三级技能** 。  
 
2. 人类挂机时，*第一技能格* 推荐使用 ***“精研百草”、“轻如鸿毛”*** ；  

3. 屠夫挂机时，*第一技能格* 推荐使用 ***“大心脏”、“人肉烧烤”、“辣手屠夫”*** ，极推荐携带 **“光明之眼”**

### 可能出现的一些问题
- 启动工具后请勿操作鼠标、键盘，耐心等待，挂机过程中出现错误，请于匹配大厅重启工具，结束工具请按F10。

- 对于游戏内角色无反应的情况，可以尝试一、二、三级技能混带

&emsp;<img src="https://github.com/maskrs/DBD_AFK_TOOL/blob/main/image-foder/%E6%8A%80%E8%83%BD3.png" width="500px">

&emsp;&emsp;也有可能是皮肤的问题 *【军团、小丑、枯魔、护士】* 常见，可以更换皮肤重新尝试。  

- 人类无法奔跑，可能是硬件模拟被拦截，可尝试关闭Windows安全中心内核隔离。  

- 对于启动工具无反应的情况，请检查系统设置
